# Проект дипломной работы профессии «Python-разработчик с нуля»
## Backend приложения для социальной сети для обмена фотографиями

## Цель дипломной работы:
Разработка API для загрузки публикаций и их фото с возможностью комментировать и ставить лайки.

### Реализация проекта:

1. Посты состоят из текста и фотографии, сохраняется время создания поста.
2. Публикации создаются только авторизованными пользователями, редактировать публикацию может только её автор.
3. Комментарии могут быть написаны к определённой публикации, оставлять их могут только авторизованные пользователи. Сам комментарий состоит из текста и даты его публикации.
4. Пользователи могут оставлять реакцию на публикацию в виде лайка с подсчетом их количества
5. При получении деталей публикации помимо полей самой публикации отображается список комментариев и количество лайков к публикации, например:

### Для запуска проекта:

1. Установлены зависимости (pip install -r requirements.txt)
2. Созданы базы данных в postgres и соответствующие миграции (manage.py migrate, make migrations)
3. Запущен проект приложения (python manage.py runserver)
4. Созданы суперпользователи и получены токены для обеспечения доступа к приложению
